A few things that you may want to consider when working with a website. A web url has important parts

  • Protocal (http
  • Address (copper.hungrybeagle.com)
  • Directory /
  • Filename /index.html
Example: http://copper.hungrybeagle.com/files/index.html

The body of your document is where you should put all of your container tags. Note that all of the formatting is dictated by html markup tags and the styles that are applied to them.

Styles are extremely important. Styles are how we can change the look of a document. A style can even make a page look like an application on a mobile device

You should pay attention to how an html tag becomes part of a class, as well as how the classes are important for defining styles

Some of the important tags that you can use are the <p> tag, the <div> tag and the <span> tag.

The div tag creates a specific section, much like a paragraph. It will have space between the sections much like a paragraph will. What is the difference between them? Not much. It's more or less just a reference to their function

Some new tags that were included have similar function...the <section> tag and the <nav> tag do the same things, but have different names to help make tags within tags less confusing.

The <span> tag is also a container, but is intended for doing things inline instead of in a paragraph.
